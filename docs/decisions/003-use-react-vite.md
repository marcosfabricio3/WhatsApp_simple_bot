# Title

Uso de React + Vite para frontend

## Status

Accepted

## Context

Se requiere una interfaz simple para configuración y monitoreo.

## Opciones:

- HTML simple
- React
- Next.js

## Decision

Se utiliza:

- React
- Vite

## Rationale

- Desarrollo rápido
- Hot reload eficiente
- Baja complejidad (no SSR necesario)

## Consequences

### Positivas

- UI moderna
- Fácil mantenimiento

### Negativas

- Requiere separar frontend/backend

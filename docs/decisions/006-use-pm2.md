# Title

Uso de PM2 para ejecución 24/7

## Status

Accepted

## Context

El sistema debe ejecutarse continuamente.

## Decision

Se utiliza PM2 como process manager

## Rationale

- Reinicio automático
- Manejo de logs
- Fácil despliegue

## Consequences

### Positivas

- Alta disponibilidad
- Simplicidad operativa

### Negativas

- Dependencia adicional

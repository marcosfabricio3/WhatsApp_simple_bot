# Title

Visualización de logs por chat

## Objetivo

Permitir inspección detallada de actividad.

## Requisitos

- Logs filtrables por:
  - Chat
  - Fecha
  - Tipo (INFO, ERROR)
- Vista por chat individual
- Actualización en tiempo real

---

## Tipos de eventos

- Mensaje recibido
- Mensaje ignorado
- Acción ejecutada
- Error

# Title

Configuración por chat

## Objetivo

Configurar comportamiento individual por chat.

## Requisitos

Cada chat debe permitir:

- Activar/desactivar:
  - Crear evento calendario
  - Enviar email
  - Reenviar mensaje por WhatsApp/ o dar alerta
- Configurar:
  - Email destino
  - Número destino

---

## UI

- Formulario simple
- Toggles (on/off)
- Inputs claros

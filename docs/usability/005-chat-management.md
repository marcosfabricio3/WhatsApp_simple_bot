# Title

Gestión de chats

# Objetivo

Permitir crear y administrar chats monitoreados.

## Requisitos

### Sidebar

- Lista de chats configurados
- Botón “Agregar chat”

---

## Funcionalidades

- Asociar chat de WhatsApp
- Nombrar chat
- Activar/desactivar

---

## UI

- Lista vertical izquierda
- Chat activo resaltado

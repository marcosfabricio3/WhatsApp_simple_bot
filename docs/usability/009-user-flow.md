# Title

Flujo de usuario

## Flujo principal

1. Usuario abre app
2. Verifica estado (WhatsApp + IA)
3. Configura chats
4. Define acciones
5. Sistema corre automáticamente
6. Usuario monitorea logs

---

## Flujo de error

1. Indicador muestra fallo
2. Usuario usa botón reset
3. Sistema intenta reconectar

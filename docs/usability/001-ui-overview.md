# Title

Interfaz gráfica general

## Objetivo

La aplicación debe proveer una interfaz gráfica completa para configuración, monitoreo y control del sistema.

## Requisitos

- UI basada en:
  - React
  - Vite
- Debe incluir:
  - Panel principal (dashboard)
  - Sidebar de navegación
  - Vista de logs
  - Configuración por chat
  - Estado del sistema

## Layout

```
-------------------------------------
| Sidebar |        Main Panel       |
|         |                         |
| Chats   | Dashboard / Logs / etc  |
|         |                         |
-------------------------------------

```

# Title

Controles de reinicio

## Objetivo

Permitir recuperación manual ante fallos.

## Requisitos

Botones independientes:

1. Reset WhatsApp

- Reinicia sesión
- Regenera QR si es necesario

2. Reset IA

- Reinicia cliente API

3. Reset General

- Reinicia todo el sistema

## UI

- Botones claramente diferenciados
- Confirmación previa obligatoria
